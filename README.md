# 🚀 Data & Business Analyst Portfolio Website 🚀

This repository contains the source code for my personal portfolio website, designed to showcase my skills, projects, and experience as a Data and Business Analyst. The website is a comprehensive presentation of my journey, technical abilities, and the value I can bring to a data-driven organization.

## ✨ About This Portfolio

This portfolio is more than just a resume; it's an interactive experience designed to give you a deep understanding of my capabilities. I've crafted it to be a clean, modern, and user-friendly platform where you can explore my work in detail.

The primary goal of this portfolio is to demonstrate my ability to:

* **Transform data into a narrative** 📊: I bridge the gap between complex datasets and actionable business strategy.
* **Leverage key technologies** 💻: I utilize a range of tools including Python, SQL, and Power BI to deliver insights.
* **Communicate effectively** 🗣️: The portfolio itself is a testament to my ability to present information in a clear and compelling way.

## 🌟 Features

This portfolio is a single-page application with several distinct sections:

* **Hero Section:** A visually engaging introduction with my photo, title, and a brief summary of what I do.
* **About Me:** A more detailed look into my background, my passion for data, and my technical journey, starting from a foundation in electronics and moving into computer science.
* **Skills:** A dynamic and categorized display of my technical and core competencies. This includes proficiency in:
    * Data Analysis & Programming (SQL, Python, Excel/VBA)
    * Data Visualization (Power BI, Tableau)
    * Database Management (MySQL, PostgreSQL)
    * And more!
* **Projects:** A showcase of my most significant data analysis projects. Each project includes:
    * A detailed description of the project and its objectives.
    * The tools and technologies used.
    * The outcome and impact of the analysis.
    * A link to the relevant GitHub repository.
* **Certificates:** A collection of my professional certifications, demonstrating my commitment to continuous learning and skill development.
* **My Journey:** A timeline of my educational and professional experience.
* **Resume & Cover Letter:** Easy access to my professional documents, with both a preview and a download option.
* **Contact:** My contact information and links to my professional profiles on LinkedIn and GitHub.

## 🛠️ Technologies Used

This website was built using a combination of modern web technologies:

* **HTML5:** For the structure and content of the site.
* **CSS3:** For styling and layout, including a responsive design for mobile and desktop viewing.
* **JavaScript (ES6+):** For dynamic content rendering, animations, and interactive features.
* **Vanta.js:** For the animated background in the hero section.
